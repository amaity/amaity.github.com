Visualization of the DVC network in your browser.

Browser based diagram?

What will the diagram consist of?

  • Data
  • Status
  • Elements

HTML5 can do that.

  • 80MW, 132KV
  • in, out
  • bus, line, trf.

How will the diagram behave?

  • Animated
  • Scalable

SVG can do that.

What is it actually?

One single HTML file.

A HTML template.

That brings all the features needed to transform a simple HTML5 page into a view of the transmission network.

From a concept sketch ...

... to a static model ...

... to an interactive viz. finally.

Benefits and Possibilities

The immediate impact.

It will bring a subset of the DVC transmission data to every device with internet access.

It will open up new possibilities for network monitoring.

Other benefits

  • Diagnostics
  • Validation
  • Planning
  • Data

Some more benefits

  • Simple
  • It's cool
  • Hackable
  • It's current
  • Accessible

Future possibilites.

Information dashboard

Experience from the project can be utilised to build an information dashboard where the trend and profile of various plant and substation parameters can be displayed.

Some prerequisites.

This project cannot take off without
(a) access to SCADA data
(b) access to the DVC web server.

Assistance required.

Need help of the Communication Dept. for access to the Load Despatch SCADA data.

Need help of the Information Technology Dept. for access to and space on the DVC web server.

Time required to reach functionality is approximately eight months from the date data becomes accessible.
